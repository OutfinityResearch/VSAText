Characters Tab

The Characters tab manages all persons, creatures, and entities that act within your story. Well-defined characters are the foundation of compelling narratives.

When to Use This Tab

Use the Characters tab as one of your first steps when building a story. Define your protagonist, antagonist, and supporting cast before moving to structure.

What This Tab Does

The Characters tab lets you create and manage character definitions with:

  • Basic identity (name, role, type)
  • Psychological archetype
  • Personality traits
  • Character arc type
  • Flaws and weaknesses
  • Physical description

Character Properties

Property Description Values
Name Character's identifier Any string (e.g., "Anna", "TheDarkKnight")
Type Role in the narrative protagonist, antagonist, mentor, ally, character
Archetype Jungian/Campbell archetype Hero, Shadow, Mentor, Herald, Threshold Guardian, Shapeshifter, Trickster, Ally
Traits Personality characteristics courage, wisdom, cunning, loyalty, compassion, etc.
Arc Type Character's transformation positive (grows), negative (corrupts), flat (inspires others)
Flaws Weaknesses for conflict fears failure, arrogance, naivety, etc.

Character Archetypes

SCRIPTA supports archetypes from Campbell and Vogler's work:

Archetype Function Suggested Traits Examples
Hero Protagonist who transforms through trials courage, resilient, compassion Frodo, Harry Potter, Katniss
Mentor Guides and teaches with wisdom wisdom, patient, mysterious Gandalf, Dumbledore, Obi-Wan
Shadow Dark reflection, embodies rejected aspects cunning, pride, wrath Sauron, Voldemort, Darth Vader
Herald Announces the call to adventure mysterious, charismatic R2-D2, Hagrid, the White Rabbit
Shapeshifter Unpredictable ally/enemy deceit, cunning, volatile Gollum, Snape, Catwoman
Trickster Comic relief, questions status quo cunning, rebellious, creativity Loki, Fred & George, Jack Sparrow
Threshold Guardian Tests the hero before transitions stoic, strong, justice Cerberus, the Sphinx
Ally Supports the hero on journey loyalty, protective, optimistic Sam Gamgee, Ron Weasley

Trait Categories

Moral Traits

  • Positive: courage, honesty, loyalty, compassion, integrity, selflessness, justice, mercy
  • Negative: greed, cruelty, deceit, cowardice, envy, pride, wrath

Intellectual Traits

wisdom, curiosity, cunning, creativity, pragmatism, naivety

Emotional Traits

passionate, stoic, volatile, melancholic, optimistic, cynical, empathetic

Social Traits

charismatic, introverted, diplomatic, rebellious, protective, mysterious

Step-by-Step: Creating a Character

  1. Click "Add Character" button
  2. Enter the character's name
  3. Select their narrative type (protagonist, antagonist, etc.)
  4. Choose an archetype that fits their function
  5. Add 2-4 defining traits
  6. Define at least one flaw
  7. Set their arc type (how will they change?)
  8. Optionally add physical description
  9. Save the character

Metrics Impact

Characters affect several key metrics:

  • Character Consistency (CS): Do characters act according to their traits?
  • Character Arc Delivered (CAD): Does the arc type manifest in the story?
  • Character Action Ratio (CAR): Do characters have enough actions?
  • Orphan Index (OI): Are there unused characters?

Best Practices

  1. Start with protagonist: Build the story around your hero
  2. Define opposites: Antagonist should contrast protagonist
  3. Give everyone flaws: Perfect characters are boring
  4. Limit traits: 3-4 traits creates focus
  5. Match archetype to function: Don't fight the pattern
  6. Consider arc early: Transformation drives the story
