Locations Tab

The Locations tab manages places where scenes occur. Well-defined locations enhance atmosphere, provide spatial coherence, and ground your story in a believable world.

When to Use This Tab

Use the Locations tab after defining characters and before building scenes. Create the key places where your story will unfold.

What This Tab Does

The Locations tab lets you define the physical and atmospheric properties of places in your story:

  • Geographic type (urban, forest, mountain, etc.)
  • Emotional atmosphere (peaceful, threatening, mysterious)
  • Historical period (medieval, modern, futuristic)
  • Physical characteristics
  • Symbolic meaning

Location Properties

Property Description Examples

Geography Physical type urban, rural, forest, mountain, coastal, desert, underground, space
Atmosphere Emotional quality peaceful, mysterious, threatening, welcoming, desolate, sacred
Time Period Historical setting ancient, medieval, renaissance, modern, futuristic

Atmosphere Options

Choose atmospheres that support the emotional needs of scenes:

  • peaceful: Rest points, ordinary world, recovery
  • mysterious: Discovery scenes, hidden secrets
  • threatening: Danger, confrontations, ordeals
  • welcoming: Allies, safe havens, respite
  • desolate: Isolation, despair, wastelands
  • sacred: Temples, transformations, revelations
  • corrupted: Fallen places, shadow influence
  • enchanted: Magic, wonder, otherworldly

Step-by-Step: Creating a Location

  1. Click "Add Location" button
  2. Enter a descriptive name
  3. Select the geography type
  4. Choose the primary atmosphere
  5. Set the time period
  6. Add 2-3 specific characteristics
  7. Set significance level
  8. Save the location

Metrics Impact

Locations affect the Location Logic Score (locLogic):

  • Good stories reuse key locations
  • Scene-hopping without returning lowers the score
  • Locations should serve narrative purpose

Low locLogic indicates random scene placement without spatial coherence.

Best Practices

  1. Start with the ordinary world: Where does your hero begin?
  2. Create contrast: New world should differ from ordinary world
  3. Limit location count: 5-10 key locations is usually enough
  4. Return to locations: Revisiting adds meaning
  5. Match atmosphere to mood: Location supports scene emotion
  6. Add specific details: Characteristics make places memorable
