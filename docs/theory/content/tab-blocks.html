Blocks Tab

The Blocks tab manages reusable narrative building blocks - predefined templates for common story moments that can be assigned to scenes.

When to Use This Tab

Use the Blocks tab when planning scenes to identify what type of narrative moment each scene represents. Blocks help ensure variety and align with arc beats.

What This Tab Does

Narrative blocks are pre-defined story patterns that represent common narrative moments. The Blocks tab lets you:

  • Browse available block types
  • Understand block purposes and suggestions
  • Assign blocks to scenes
  • Ensure variety in scene types
  • Align scenes with arc beats

Block Categories

Opening Blocks

Blocks for story beginnings and setup.

Block Description Suggested Moods
ordinary_world Establish the hero's normal life before adventure peaceful, melancholic
call_to_adventure Something disrupts normality and invites action mysterious, adventurous
refusal_of_call The hero initially resists due to fear tense, melancholic

Transition Blocks

Blocks for moving between story phases.

Block Description Suggested Moods
crossing_threshold The hero commits to the adventure adventurous, tense
meeting_mentor The hero encounters a guide mysterious, peaceful
tests_allies_enemies The hero faces challenges and learns trust tense, adventurous

Confrontation Blocks

Blocks for major challenges and conflicts.

Block Description Suggested Moods
approach_cave The hero prepares for major challenge ominous, tense
ordeal The hero faces their greatest fear desperate, epic
death_rebirth Symbolic death and rebirth desperate, revelatory

Resolution Blocks

Blocks for wrapping up the story.

Block Description Suggested Moods
reward The hero claims the prize triumphant, peaceful
road_back The hero must return tense, adventurous
resurrection Final test using everything learned epic, triumphant
return_elixir Hero returns home transformed peaceful, bittersweet

Micro Blocks

Smaller narrative moments that can occur anywhere.

Block Description Suggested Moods
revelation A hidden truth changes everything revelatory, mysterious
betrayal A trusted ally reveals hostile nature desperate, tense
sacrifice A character gives up something precious bittersweet, epic
reunion Separated characters come back together triumphant, intimate
confrontation Two opposing forces face each other tense, epic
escape Characters flee from danger tense, desperate
discovery Something new and important is found mysterious, adventurous
temptation Character faces choice testing values tense, intimate
rescue One character saves another epic, triumphant
loss Character loses something precious melancholic, desperate
transformation Character undergoes significant change revelatory, epic
choice Character must decide with major consequences tense, intimate

Assigning Blocks to Scenes

  1. Navigate to a scene in the structure view
  2. Click "Assign Block"
  3. Select the appropriate block type
  4. The block's suggested mood can be applied automatically

Metrics Impact

  • Scene Variety: Using diverse block types improves pacing
  • Beat Alignment: Blocks should match arc beat positions
  • Mood Consistency: Block mood suggestions help coherence

Best Practices

  1. Don't overuse blocks: Not every scene needs a dramatic block
  2. Match to arc: Opening blocks early, resolution blocks late
  3. Vary intensity: Alternate high-intensity and calm blocks
  4. Use micro blocks: They add texture anywhere
  5. Follow mood suggestions: Blocks know their emotional needs
