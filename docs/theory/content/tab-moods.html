Moods Tab

The Moods tab defines emotional atmospheres for scenes. Moods combine multiple emotions with varying intensities to create the feeling you want readers to experience.

When to Use This Tab

Use the Moods tab when planning scene emotions. Select preset moods or create custom emotional palettes for specific story moments.

What This Tab Does

The Moods tab provides:

  • Preset mood templates for common story moments
  • Custom mood creation with emotion mixing
  • Intensity controls for emotional weight
  • Mood assignment to scenes
  • Visual emotional palette display

Creating Custom Moods

Combine emotions with intensity weights (1-3):

  1. Click "New Mood"
  2. Name your mood
  3. Add emotions from the palette
  4. Set intensity for each (1=subtle, 2=present, 3=dominant)
  5. Preview the emotional mix
  6. Save the mood

Metrics Impact

Moods affect the Emotional Arc Profile (EAP):

  • The sequence of moods across scenes should align with the tension curve
  • Mood variety prevents emotional monotony
  • Mood transitions should be smooth (unless intentionally jarring)

Mood-Scene Alignment

Match moods to narrative beats:

Return bittersweet, peaceful

Best Practices

  1. Start with presets: Use proven combinations first
  2. Create contrast: Alternate emotional intensities
  3. Match location: Mood should complement setting
  4. Support tension curve: Follow your blueprint design
  5. Limit custom moods: Too many becomes unwieldy
  6. Name descriptively: "PreBattleDread" not "Mood7"
