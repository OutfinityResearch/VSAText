Objects Tab

The Objects tab (also called Plot Elements) manages physical items, artifacts, documents, and symbols that drive the plot or carry thematic weight.

When to Use This Tab

Use the Objects tab when you have items that will be important to your plot - MacGuffins, magical artifacts, key documents, or symbolic objects.

What This Tab Does

Objects in SCRIPTA serve multiple narrative functions:

  • Drive the plot forward (quests, heists)
  • Reveal information (documents, prophecies)
  • Enable actions (weapons, tools)
  • Symbolize themes (metaphorical objects)
  • Connect characters (gifts, heirlooms)

Step-by-Step: Creating an Object

  1. Click "Add Object" button
  2. Enter a memorable name
  3. Select the object type
  4. Set its significance level
  5. Optionally assign an owner
  6. Write a brief description
  7. Save the object

Chekhov's Gun Principle

SCRIPTA tracks object usage to enforce narrative economy:

Chekhov's Gun

"If you say in the first chapter that there is a rifle hanging on the wall, in the second or third chapter it absolutely must go off."

Objects defined with high significance should appear in scenes and affect the plot. The Object Integration (OI) metric tracks this.

Metrics Impact

Objects affect several metrics:

  • Object Integration (OI): Are important objects used in the plot?
  • Orphan Index: Unused objects lower this score
  • Setup/Payoff: Objects should be introduced before becoming crucial

Best Practices

  1. Limit important objects: 3-5 high-significance objects per story
  2. Introduce early: Important objects should appear before needed
  3. Track ownership: Know who has what at each point
  4. Match type to function: Use appropriate categories
  5. Use objects for connection: Gifts link characters
  6. Consider symbolic weight: Objects can carry theme
