Relations Tab

The Relations tab defines connections between characters. Relationships drive plot dynamics, create conflict, and provide emotional stakes.

When to Use This Tab

Use the Relations tab after defining your characters. Map out how they connect to each other before building your story structure.

What This Tab Does

The Relations tab manages the web of connections between your characters:

  • Family bonds (siblings, parent-child, spouses)
  • Social connections (friends, rivals, enemies)
  • Hierarchical relationships (mentor-student, master-servant)
  • Romantic entanglements (lovers, unrequited love)
  • Secret connections (hidden alliances, blackmail)
  • Conflict relationships (betrayal, revenge, nemesis)

Relationship Properties

Property Description Example
Source The character initiating/defining the relationship Anna
Target The other character Marcus
Type Nature of the connection siblings, mentor_student, lovers
Bidirectional Whether relationship works both ways siblings (yes), unrequited (no)
Hidden Secret relationship not known to others secret_allies, blackmail

Step-by-Step: Creating a Relationship

  1. Click "Add Relationship" button
  2. Select the source character
  3. Select the target character
  4. Choose the relationship type
  5. The system shows if it's bidirectional
  6. Optionally mark as hidden (for secrets)
  7. Save the relationship

Metrics Impact

Relationships affect the Relationship Usage Score (relUsage):

  • Defined relationships should manifest in scenes
  • Characters in relationships should appear together
  • Relationship types should influence interactions

Low relUsage indicates "decorative" relationships that don't impact the story.

Visualization

The Relations tab includes a visual graph showing:

  • Characters as nodes
  • Relationships as colored edges
  • Relationship types indicated by color
  • Direction arrows for non-bidirectional relationships

Best Practices

  1. Connect your protagonist: Hero should have relationships to key characters
  2. Create triangles: A-B-C relationships create rich dynamics
  3. Include conflict: At least one antagonistic relationship
  4. Use hidden relationships: Great for plot twists
  5. Balance positive/negative: Mix supportive and adversarial
  6. Make relationships active: Relationships should drive scenes
